# Bitsy-Web-Gallery
This is a simple html based web gallery for Bitsy games. It is primarily focused on making playing Bitsy games on a mobile device more fun. I have added in a simple joypad because I found using touch in a Bitsy game on a mobile device to be fussy. It also has the advantage in that it makes Bitsy games easier to play for those with hand dexterity issues.
Put all three files in your directory on your web server. Create two directories, games and thumbs. Put your games in the games folder, thumbnails for your games in thumbs. The gallery will scale down your screen captures to fit. If you don't have a thumbnail for a game you get a generic gamepad icon.
Edit script.js to add your games and thumbnails. Remember to not have spaces in the file names for your thumbnails and Bitsy html files.
Note that the joypad will only work correctly on a web server, not locally - unless you are running a web server locally.
Chill and enjoy.
